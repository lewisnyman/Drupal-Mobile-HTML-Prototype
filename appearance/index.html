Appearance

Set and configure the default theme for your website. Alternative themes are available.

Enabled themes

  • Bartik
    A flexible, recolorable theme with many regions. Currently the default theme for all users.
  • Seven
    A simple one-column, tableless, fluid width administration theme.

Disabled themes

  • Garland
    A multi-column theme which can be configured to modify colors and switch between fixed and fluid width layouts.
  • Stark
    This theme demonstrates Drupal's default HTML markup and CSS styles.
